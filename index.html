Simple Backup Utility

The easiest way to zip and archive your files and folders on Linux.

1. Download the Package

Download the latest Debian package (`.deb` file) directly from our GitHub Releases page.

Download Latest .DEB

2. Install the Utility

After downloading, run the following command in your terminal. 

                        
                            sudo apt install zip tar -y
                            sudo dpkg -i BACKUP.deb

Command copied to clipboard!

Usage Guide

The utility is run using the simple `backup` command. Provide the directory to archive and a name for the output file.

backup [source_directory]

// Example: Back up the 'documents' folder into 'documents.zip'

backup /home/user/documents

Key Features

**Regular updates!** We are always improving the tool with new features and fixes.

**One-Command Archiving:** Easily create compressed archives with a single, clear command.

**Timestamped Backups:** Automatically organizes your archives by date and time.

**TAR & ZIP Compression:** Uses standard Linux tools (`tar` and `zip`) for compact storage.

**Simple Interface:** Designed to be straightforward for quick, hassle-free backups.

**Auto Folder Creation:** Puts all the files in a folder name backups that it automatically creates!

Frequently Asked Questions (FAQ)

Which operating systems are supported?

The Simple Backup Utility is packaged as a **.deb** file, meaning it is designed for Debian-based Linux distributions, including **Ubuntu, Mint, Kali, and Debian itself**.

How do I uninstall the utility?

You can easily remove the utility using the standard package manager command: sudo dpkg -r backup

Where does the utility save the archived files?

By default, all archives are saved in a folder named backups located in the directory where you run the `backup` command.

Does this tool require an internet connection?

No. Once the initial package is downloaded and installed, the utility uses only local system tools (`zip` and `tar`) and operates completely offline.
